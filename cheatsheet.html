Configuring git

Setting your name and email:

git config --global --add user.name "Jelmer van der Linde"
git config --global --add user.email jelmer@ikhoefgeen.nl

Adding and committing

Adding and committing test.html:

git add text.html
git commit -m "Updating the test text."

Seeing the status of your current working directory:

git status

Seeing your changes:

git log

Branches

See all branches and your current branch:

git branch

Creating a branch:

git branch branch-name

Switching to a branch:

git checkout branch-name

Merging an other branch into your current branch:

git merge other-branch

Deleting a branch:

git branch -d branch-name

Pushing and pulling

Getting started by cloning a repository:

git clone your-remote-repo.git target-directory

Listing all your remotes:

git remote -v

Pushing to remote:

git push [remote-name] [branch-name]

Pulling from remote:

git pull [remote-name]
